## 📌 Project Overview

ArtifyAI is a web-based digital art gallery platform integrated with an emotion recognition system. It allows users to explore artworks and enhances user experience by analyzing facial emotions to provide more personalized interactions.

The system was initially built using a local deep learning model (`model.h5`) and later upgraded to use an external API for improved scalability and performance.

## 🚀 Features

* 🎨 Digital art gallery with categorized artworks
* 😊 Emotion detection (Joy, Sadness, Calmness, Anger, Fear)
* 🧠 AI-based user experience enhancement
* 📷 Image-based emotion analysis
* 🗂️ Organized artwork categories:

  * Pottery
  * Sculptures
  * Textile Arts
  * Portraits
  * Animal Paintings
  * Landscapes
  * Flowers
  * Abstract
* 📩 Contact / inquiry system
* 🛒 Order management system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Flask (Python)
* **Database:** Neon PostgreSQL
* **AI Model:** MobileNetV2 (Emotion Recognition – initial version)
* **Explainability Tools:** Grad-CAM, LIME
* **Current Integration:** Artemis API (replacing local `.h5` inference)
